## Descripción del Proyecto
Proyecto de análisis visual desarrollado en **Tableau Public** para analizar el desempeño académico de estudiantes y comprender los factores asociados a su rendimiento.

El proyecto permite explorar resultados académicos y brechas de desempeño a partir de variables como **asistencia, hábitos de estudio, acceso a internet y contexto familiar**, utilizando dashboards interactivos y data storytelling.

## Dataset
**Fuente:** https://www.kaggle.com/datasets/kundanbedmutha/student-performance-dataset

El dataset contiene aproximadamente **10.000 registros a nivel estudiante**, con las siguientes variables principales:

- **Identificación:** student_id  
- **Demográficas:** age, gender, school_type  
- **Contexto familiar:** parent_education, internet_access  
- **Hábitos académicos:** study_hours, study_method, extra_activities  
- **Logística y asistencia:** travel_time, attendance_percentage  
- **Resultados académicos:**  
  - math_score  
  - science_score  
  - english_score  
  - overall_score  
  - final_grade  

## Preguntas de Análisis Abordadas
1. ¿Cuál es el nivel general de desempeño académico de los estudiantes?  
2. ¿Qué factores se asocian con mejores resultados académicos?  
3. ¿Qué tan relevante es la asistencia frente a otros hábitos de estudio?  
4. ¿Existen brechas de desempeño según el contexto educativo y social?  
5. ¿Cómo influyen el acceso a internet y la educación parental en los resultados?

## Principales Insights

### Resumen Ejecutivo
- El desempeño académico promedio es moderado, con diferencias claras entre grupos.
- La **asistencia** presenta una relación más consistente con el rendimiento que las horas de estudio aisladas.
- Factores contextuales muestran mayor impacto que variables individuales.

### Asistencia
- Estudiantes con **alta asistencia** obtienen mejores puntajes promedio en todas las materias.
- La baja asistencia amplifica las brechas de desempeño académico.

### Acceso a Internet
- El acceso a internet se asocia con mayor asistencia y mejores resultados académicos.
- La combinación **sin internet + baja asistencia** representa el escenario de mayor riesgo.

### Educación Parental
- Un mayor nivel educativo parental se relaciona con mejores resultados.
- Sin embargo, una buena asistencia puede mitigar parcialmente estas brechas.

### Equidad
- Las brechas de desempeño no responden a un único factor, sino a la acumulación de desventajas contextuales.
- El análisis destaca oportunidades de intervención enfocadas en asistencia y acceso.

## Conclusiones & Recomendaciones
1. La asistencia es el factor más consistente asociado al desempeño académico.  
2. Mejorar el acceso a internet puede tener impacto indirecto en los resultados.  
3. Las intervenciones educativas deberían priorizar contextos de mayor vulnerabilidad.  
4. El análisis se enfoca en asociaciones observables y evita conclusiones causales.

## Estructura del Dashboard
- **Dashboard 1:** Visión General — distribución de puntajes, materias y calificaciones finales.  
- **Dashboard 2:** Factores Clave — asistencia, hábitos y métodos de estudio.  
- **Dashboard 3:** Brechas & Contexto — equidad, acceso a internet y educación parental.
